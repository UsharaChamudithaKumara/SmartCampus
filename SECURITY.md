# Security Policy

## Supported Versions

Currently, only the latest version of the Smart Campus Operations Hub is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Instead, report it to the maintainers:

1.  Email the lead developer at `security@smartcampus.edu` (Note: This is a placeholder).
2.  Provide a detailed description of the vulnerability.
3.  Include steps to reproduce the issue.

We will acknowledge your report within 48 hours and provide a timeline for a fix.
